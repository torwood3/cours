Interface web avancée 2016

Ce cours d'IWA a pour but de vous fournir le nécessaire pour vous rendre autonome dans la création d'une application web dynamique (RIA). Une "Rich internet application" est une application qui possède les fonctionnalités avancées d'un client lourd (logiciel à installer localement) et les avantages d'un client léger (interface s'exécutant dans le navigateur couplé à un serveur).

Ojectifs ?

Comprendre l'évolution et les enjeux des technologies web ainsi qu’avoir une vue générale des technologies

Savoir utiliser des bibliothèques et des frameworks front-end

Être capable de construire une application web dynamique (RIA) à partir de ces technologies.
